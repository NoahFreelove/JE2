JE2 Documentation

Keyboard

public class Keyboard

Summary

The keyboard class facilitates all keyboard input for the engine. You can check if specific keys are pressed or add events for when any keys are pressed/released.

Methods

public static void addKeyPressedEvent(KeyPressedEvent event)

Adds a key pressed event.

public static void addKeyReleasedEvent(KeyReleasedEvent event)

Adds a key released event.

public static void removeKeyPressedEvent(KeyPressedEvent event)

Removes a key pressed event.

public static void removeKeyReleasedEvent(KeyReleasedEvent event)

Removes a key released event.

public static void triggerKeyPressed(int code, int mods)

Simulates a key press

public static void triggerKeyReleased(int code, int mods)

Simulates a key release

public static void triggerKeyRepeat(int code, int mods)

Simulates holding a key down

public static void triggerKey(int code, int mods)

Simulates a key press and release

private static void onPaste(String str)

Sends clipboard info to the nuklear library

public static void pasteClipboard()

Paste current clipboard content

public static void copyToClipboard(String str)

Send a string to the clipboard`

private static void keyPressed(int code)

Updates keys array at the given code index to true

private static void keyReleased(int code)

Updates keys array at the given code index to false

public static boolean within(int min, int max, int v)

returns true if v is within the range of min and max

public static boolean isKeyPressed(int code)

returns true if the key code is pressed

public static boolean isComboPressed(KeyCombo combo)

returns true if the key combo is pressed

public static int nameToCode(String keyName)

returns the code of the key name. Ex. A->65

public static String codeToName(int keyCode)

returns the name of the key code. Ex. 65->A

Fields

public static boolean ignoreCopyPasteKeys

Set to true if you would like to ignore ctrl+c and ctrl+v inputs

public static boolean disableInput

Set to true if you would like to ignore all keyboard input. For example if you don't want the player to move during loading sequences you would set this to true.

private static final ArrayList<KeyPressedEvent> keyPressedEvents

Arraylist of all key pressed events. All events will be triggered every key press.

private static final ArrayList<KeyReleasedEvent> keyReleasedEvents

Arraylist of all key pressed events. All events will be triggered every key press.

private static final boolean[] keys

1024 booleans that hold the status of a given keycode being pressed. Ex. index 65 being true would corrospond to A being pressed.
