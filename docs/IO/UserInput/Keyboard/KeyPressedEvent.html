JE2 Documentation

Keyboard

KeyPressedEvent

Interface

Summary

The KeyPressedEvent interface lets you add key press events to the Keyboard class. These events will be invoked every time a key is pressed.

Methods

void invoke(int key, int mods)

key - the keycode to invoke. mods - any modifiers like ctr, alt, shift.

Fields

This interface has no fields.

Other Info

No additional info.
