JE2 Documentation

Keyboard

KeyReleasedEvent

Interface

Summary

The KeyReleasedEvent interface lets you add key release events to the Keyboard class. These events will be invoked every time a key is released.

Methods

void invoke(int key, int mods)

key - the keycode to invoke. mods - any modifiers like ctr, alt, shift.

Fields

This interface has no fields.

Other Info

No additional info.
