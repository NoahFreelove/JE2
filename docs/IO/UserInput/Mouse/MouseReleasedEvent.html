Mouse

public interface MouseReleasedEvent

Interface

Summary

The MouseReleasedEvent interface lets you add mouse released events to the Mouse class. These events will be invoked every time a mouse button is released.

Methods

void invoke(MouseButton button, int mods)

button - the mouse button to invoke.
mods - any modifiers like ctr, alt, shift.

Fields

This interface has no fields.

Other Info

No additional info.
