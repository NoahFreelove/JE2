JE2 Documentation

Mouse

public class Mouse

Summary

The mouse class handles all mouse inputs for the engine. You can add events for mouse pressed and releases as well as check if a specific button is pressed. The mouse class also facilitates the mouse position relative to the window and in world space.

Methods

public static void addMousePressedEvent(MousePressedEvent event)

Adds a mouse press event

public static void addMouseReleasedEvent(MouseReleasedEvent event)

Adds a mouse release event

public static void removeMousePressedEvent(MousePressedEvent event)

Removes a mouse press event

public static void removeMouseReleasedEvent(MouseReleasedEvent event)

Removes a mouse release event

public static void triggerMouseMoved(float x, float y)

Sets the x and y position of the mouse

public static Vector2f getMousePosition()

returns the position of the mouse relative to the window

public static float getX()

returns the x position of the mouse relative to the window

public static float getY()

returns the y position of the mouse relative to the window

public static void triggerMousePressed(MouseButton button, int mods)

Simulates a mouse press

public static void triggerMouseReleased(MouseButton button, int mods)

Simulates a mouse release

public static void triggerMouseClick(MouseButton button, int mods)

Simulates a mouse press and release

private static void mousePressed(int button)

Sets the button array at index button to true

private static void mouseReleased(int button)

Sets the button array at index button to false

public static boolean isPressed(int button)

returns true if the button is pressed

public static int nameToCode(String buttonName)

Converts a name to a button code. Ex. "LEFT" -> 0

public static Vector2f getMouseWorldPosition2D()

returns getMousePosition2D(Camera c) with the currently active scene camera.

public static Vector2f getMouseWorldPosition2D(Camera c)

Gets the mouse's position in world space. This is achieved through matrix transformations. Somewhat expensive to call.

Fields

private static float x

X position of mouse.

private static float y

Y position of mouse.

public static boolean disableInput

Set this to true if you'd like to disable mouse button input.

private static final ArrayList<MousePressedEvent> mousePressedEvents

Collection of all mouse press events. All will run whenever a mouse button is pressed.

private static final ArrayList<MouseReleasedEvent> mouseReleasedEvents

Collection of all mouse release events. All will run whenever a mouse button is released.

private static final boolean[] buttons

Holds the press states of mouse buttons. Ex. index 0 is mouse button 0's press state
