JE2 Documentation

Mouse

MousePressedEvent

Interface

Summary

The MousePressedEvent interface lets you add mouse press events to the Mouse class. These events will be invoked every time a mouse button is pressed.

Methods

void invoke(MouseButton button, int mods)

button - the mouse button to invoke. mods - any modifiers like ctr, alt, shift.

Fields

This interface has no fields.

Other Info

No additional info.
