Audio

public sealed class AudioSource extends Script permits AudioSourcePlayer

Summary

The audio source manages the audio resource (ex. the byte buffer) and the OpenAL playback. The audio source cannot be directly instantiated, but must be created by through the AudioSourcePlayer.

Constructors

protected AudioSource()

returns a new instance of AudioSource

protected AudioSource(Resource<AudioBundle> bundleResource)

returns a new instance of AudioSource with a given audio bundle resource

Methods

public AudioSource setAudio (Resource<AudioBundle> resource)

Sets the audio resource for this audio source. And regenerates the audio buffer.

private void generateAudioBuffer()

Generates the OpenAL audio buffer and removes any audio filters.

protected void delete ()

Deletes the audio source.

protected void playSound ()

Plays audio from beginning.

protected void playAt (int pos)

Plays the audio source at the given position. Note: Position is in bytes, not seconds.

protected void stopSound ()

Stops the audio source.

public boolean isPlaying ()

Returns true if the audio source is playing.

public boolean loops ()

Returns true if the audio source is set to loop.

public void setLoops (boolean loops)

Sets whether the audio source should loop.

public void setFilter (SoundFilter filter)

Sets the audio filter for this audio source.

public void updateFilter ()

Updates the audio filter for OpenAL.

public SoundFilter getFilter ()

Returns the audio filter for this audio source.

public void removeFilter ()

Removes the audio filter for this audio source.

public void setGain (float gain)

Sets the gain for this audio source.

public void setMaxGain (float maxGain)

Sets the max gain for this audio source.

public int getBufferPosition ()

Returns the current position of the audio buffer.

public void setAttribute3f (int attribute, Vector3f value)

attribute - The attribute to set.
value - The value to set the attribute to.

public int getSourceSize ()

Returns the size of the audio source in bytes.

public float getDuration ()

Returns the duration of the source in seconds.

public float getPositionTime ()

Returns the position of the audio source in seconds.

public float duration ()

Returns the duration of the source in seconds.

public float getDecimal ()

Returns the percentage of the audio source that has been played from 0 to 1. Useful for progress bars perhaps?

(package-private) class AudioProcessor

Summary

The AudioProcessor class is used to load audio files and convert them to a format that can be played by an AudioSource.

Methods

public static AudioBundle processAudio (String filePath)

Loads the audio file at the given path and returns an AudioBundle containing the audio data.

public static AudioBundle processAudio (byte[] data)

Loads the audio from the given byte array and returns an AudioBundle containing the audio data.
