JE2 Documentation

Audio

public class Soundtrack

Summary

The soundtrack is primarily for world ambient sound or background music. You provide it a track list, and it will play sequentially. It will also loop the track list when it reaches the end.

Constructors

public Soundtrack(AudioSourcePlayer... tracks)

returns a new instance of Soundtrack with all the tracks provided.

Methods

public void addToWorld()

Adds soundtrack to the active scene.

public void addTrack(AudioSourcePlayer track)

Adds a track to the soundtrack.

public void removeTrack(AudioSourcePlayer track)

Removes a track from the soundtrack.

public void removeTrack(int track)

Removes a track from the soundtrack at the given index.

public void playTrack(int index)

Plays track at given index.

public void start()

Starts the soundtrack.

public void nextTrack()

Skips to the next track.

Fields

private CopyOnWriteArrayList<AudioSourcePlayer> trackList

The list of tracks.

private int currentTrack

Index of the current track.

private AudioSourcePlayer activeTrack

The reference to the current track.

private Watcher trackEndWatcher

This watcher will check every frame for when the current track is over and will skip to the next track.
