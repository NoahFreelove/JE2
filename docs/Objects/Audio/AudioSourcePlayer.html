JE2 Documentation

Audio

public non-sealed class AudioSourcePlayer extends AudioSource

Summary

An attachable script which plays an audio source. Tied to a scene and (should)will stop playing when the scene is unloaded. When trying to create a new AudioSourcePlayer use the checkExistElseCreate method. Do not use the default constructor.

Constructors

public AudioSourcePlayer()

returns a new instance

private AudioSourcePlayer(Resource<AudioBundle> resource, boolean newResource)

returns a new instance with a loaded resource

Methods

public static AudioSourcePlayer checkExistElseCreate(String name, int ID, String bytePath)

This method will check if the audio resource already exists to save time and memory. If not, it will create a new resource and give it to the player.
name - the resource name, make a new one if it doesn't exist

ID - Set to -1 if you don't know what to reference, otherwise enter the ID

bytePath - the relative path to the audio file from your class loader resource folder.(by default I believe its src/main/resources/your_relative_path)

public void play()

Plays the audio source

public void stop()

Stops the audio source

public void restart()

Plays the audio source from the beginning.

@Override public void destroy()

Stops the sound and removes it from the world when the parent object is destroyed.

@Override public void unload(Scene oldScene, Scene newScene)

Stops the sound when a scene change is detected.

@Override public void gameObjectAddedToScene(Scene scene)

Adds the audio source to the scene when the parent object is added to the scene.

Fields

The world the audio source is in.
