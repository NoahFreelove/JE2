CameraEffects

public class CameraShake extends Script

Summary

The CameraShake script is used to shake the camera viewport. This is not a shader effect, but rather a script that moves the camera viewport.

Constructors

public CameraShake()

returns a new instance of CameraShake.

Methods

@Override public void update()

Shake the camera viewport every frame.

@Override public void setActive(boolean newState)

When the scripts active state changes we don't want to keep shaking the camera.

Fields

public float magnitude

The magnitude of the shake.

public Camera cameraReference

The reference to the camera to shake.
