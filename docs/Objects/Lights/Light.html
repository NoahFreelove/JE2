JE2 Documentation

Lights

public abstract class Light extends Script

Summary

The light class is the base class for all lights. It is an abstract class and cannot be instantiated. Lights only affect objects which have a shader that supports lighting.

Constructors

public Light(Color color, float intensity, int type)

returns a new instance of Light.

Methods

@GLThread public final void setLighting(ShaderProgram shaderProgram, int index)

Sets the base light shader properties like position, color, intensity, and type.

@GLThread public final void hideLighting(ShaderProgram shaderProgram, int index)

Sets the light properties that make it invisible.

@GLThread protected abstract void setLightSpecific(ShaderProgram shaderProgram, int index)

For use by subclasses to set light specific properties.

@Override public void gameObjectAddedToScene(Scene scene)

Adds the light to the scene when the parent game object is added to the scene.

@Override public void start()

Adds the light to the scene when the parent game object starts.

public Light setColor(Color c)

Sets the color of the light.

Fields

public Vector2f boundPos

The position of the bound in 2d space.

public Vector2f boundSize

The size of the bound.
