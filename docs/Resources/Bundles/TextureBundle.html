JE2 Documentation

Bundles

public class TextureBundle extends ResourceBundle

Summary

The TextureBundle is a resource bundle made for image sources. It uses a ByteBuffer for the image data but also has a field for image size.

Constructors

public AudioBundle()

returns a new instance of AudioBundle with non-null fields.

public AudioBundle (ShortBuffer soundData, int format, int sampleRate, int channels)

returns a new instance of AudioBundle with set data.

Methods

public Vector2i getImageSize()

Returns the width and height of the loaded image data.

public ByteBuffer getImageData()

Returns the image data as a byte buffer.

@Override public boolean compareData(byte[] input)

Checks if this texture bundle has the same data as input.

@Override public byte[] getData()

Returns byte buffer data in byte array format

@Override public void freeResource()

Sets image data and image size to null.

Fields

private transient Vector2i imageSize

The size of the image as 2 ints. x - width. y - height.

private transient ByteBuffer imageData

The stored byte data in a buffer format.

private transient String filepath

The texture's filepath. May be empty if bundle was set with existing data.
