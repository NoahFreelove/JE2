Bundles

public abstract class ResourceBundle implements Serializable

Summary

The resource bundle is what actually holds the bytes of data for a given resource type. All resources require a bundle otherwise they are empty. Resource bundles are abstract as different file types like audio and images will have different needs. All resource bundles have a filepath or a bytePath (filepath using class loader) and a isLoaded property. Bundles are responsible for the freeing of their resources.
