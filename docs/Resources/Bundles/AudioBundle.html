Bundles

public class AudioBundle extends ResourceBundle

Summary

The AudioBundle is a resource bundle made for audio sources. It uses a ShortBuffer for the sound data and contains important audio info like format, sampleRate and channels.

Constructors

public AudioBundle()

returns a new instance of AudioBundle with non-null fields.

public AudioBundle (ShortBuffer soundData, int format, int sampleRate, int channels)

returns a new instance of AudioBundle with set data.

Methods

@Override public boolean compareData(byte[] input)

Checks if this audio bundle has the same data as input.

@Override public byte[] getData()

Returns short buffer data in byte format

@Override public void freeResource()

Resets all audio bundle fields to 0 or null.

public ShortBuffer getSoundData()

Returns sound buffer data.

public int getFormat()

Returns the audio format.

public int getSampleRate()

Returns the audio sample rate.

public int getChannels()

Returns the audio channels.

Fields

private transient ShortBuffer soundData

The audio's data is stored in this Short Buffer. OpenAL plays this back.

private transient int format

The audio source's format.

private transient int sampleRate

The audio source's sampleRate. (Number of samples per second in the audio.)

private transient int format

The audio source's channels. (If it's mono, stereo, etc.)
