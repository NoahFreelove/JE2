Bundles

public class DefaultBundle extends ResourceBundle

Summary

The Default bundle is essentially a "default" way to store data. It simply has a byteBuffer as its data which makes it easy to do operations with.

Constructors

public DefaultBundle()

returns a new instance of DefaultBundle with a 1 byte buffer.

public DefaultBundle (ByteBuffer data)

returns a new instance of DefaultBundle with set data.

public DefaultBundle (String dataPath)

returns a new instance of DefaultBundle and loads data from DataLoader.

Methods

@Override public boolean compareData(byte[] input)

Checks if this bundle has the same data as input.

@Override public byte[] getData()

Returns byte buffer converted to a byte array.

@Override public void freeResource()

Sets data to null.

Fields

private ByteBuffer data

The resource's byte data.
