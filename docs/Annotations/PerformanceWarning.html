Annotations

PerformanceWarning

Modifiers:
public

Summary

The PerformanceWarning enum is to be applied to methods which when run cause a noticeable performance hit.

Methods

int Severity()

returns severity of performance hit (lower is better).

String Reason()

returns the reason for why it is not performant.

Fields

This annotation has no fields
