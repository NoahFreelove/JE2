JE2 Documentation

Annotations

ActPublic

Modifiers:
public

Summary

The ActPublic enum should be applied to fields which are private or protected that you wish to appear visible to the JE2 Object Parser. The Object Parser is used to save and load objects. If you don't use it this annotation has little purpose.

Methods

This annotation has no methods

This annotation has no fields
