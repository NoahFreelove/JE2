# JEngine 2.0
An OpenGL remake of JEngine, a JavaFX based game engine.

Still lacking key features like a GUI, but it's getting there. 

(Uses much less memory than the JavaFX version).

## Features (so far)
- 2D vertex rendering
- Shader loading
- Texture loading
- Simple lighting

## Planned Features
- GUI
- Materials
- Physics
- Audio
- 3D Rendering
- Networking?
